Flags

Warning: All of these options are experimental.
That means they may not do what they say they will, and they may even prevent the site from working.
Proceed with caution. Flags only apply once the page has been reloaded.


Disable HTML5 History API

Force the site to use the legacy code designed for browsers that do not support the History API.

Force Four-Column Layout

Force the site to put the content into Four Columns, rather than the default maximum layout of three.

Show Advanced Options

Show the Advanced section on the options page. Contains a link to the flags page and nearly-finalized features.

Show Searchbox on Certain Pages

Show a searchbox on the flags page to allow easy finding of content.

Two-Part Toolbar

Seperate the two rows of the toolbar, putting the navigation buttons on the bottom.

Simple Toolbar

Hides the top row of the toolbar, showing only the navigation parts. Designed for bottom navigation bar.

Do not use colour on Navigation Bar

ENABLE TWO-PART TOOLBAR FIRST. Makes the navigation bar match the background colour.

Custom Themes Support

Allow a JSON file to be loaded as a theme. The JSON can define one or more themes.

TXT Support in News Popout

Allows the news popout to load a TXT file.

Enable WebAssembly

Enable the use of WebAssembly on the site.

Disable Swipe Gestures

Turn off the Swipe Gestures to navigate the site. WARNING: this opiton may break the site.
